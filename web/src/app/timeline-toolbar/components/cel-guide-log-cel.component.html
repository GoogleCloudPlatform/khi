Log CEL expressions filter individual logs that appear in the timelines. Logs that do not match the expression are grayed out.

Examples

Filter by log severity level equal or greater than WARNING severity >= WARNING
Filter by nested operation ID in the log body body('operation.id', 'operation-123')
Filter by summary containing 'error' (case-insensitive) summary.contains('error')

Helper Functions

Function Description
body(value)
B(value)		 Checks if any key or value in the body contains the pattern case-insensitively.
body(key, value)
B(key, value)		 Checks if a specific dot-separated path in the body (e.g. operation.id) matches the pattern.

Variables Available

Variable Type Description
logType string Type of log (e.g. audit, event).
severity int Severity level (e.g. INFO, WARNING, ERROR, FATAL).
summary string A brief text summary of the log.
body map The parsed structured JSON body of the log.
bodyYAML string The entire log body represented as YAML.
